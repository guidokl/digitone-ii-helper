# Elektron Digitone II — Shortcut & Key Combination Reference

---

## Modes

| Shortcut | Action |
|---|---|
| [REC] + [PLAY] | Enter Live Recording mode |
| [REC] + [PLAY], [PLAY] | Toggle Live Recording quantized/unquantized |
| [REC] + [STOP] | Enter Step Recording mode |
| [REC] + [STOP], [STOP] | Toggle Step Recording type (Standard/Jump) |
| [FUNC] + [TRK] | Enter/exit Global Mute mode (green) |
| [FUNC] + double-press [TRK] | Enter Pattern Mute mode (magenta) |
| [FUNC] + [PRESET/KIT] | Toggle Perform Kit mode |
| [FUNC] + [UP]/[DOWN] | Select Trig mode (Tracks/Velocities/Retrigs/Preset Pool) |
| [FUNC] + [AMP], D-param | Toggle Euclidean mode on/off |
| [FUNC] + D-param (EUC off) | Convert Euclidean trigs to regular trigs (hold FUNC while disabling) |
| [SONG] + [TRIG 1–16] | Enter Song mode |
| [PTN] + [TRIG 1–16] | Exit Song mode, return to pattern play |

## Copy / Paste / Clear

The universal pattern is: **[CONTEXT] + [REC]** = copy, **+ [STOP]** = paste, **+ [PLAY]** = clear.
Repeat a paste or clear immediately to undo it.

| Shortcut | Action |
|---|---|
| **Pattern (outside Grid Rec)** | |
| [FUNC] + [REC] / [STOP] / [PLAY] | Copy / Paste / Clear active pattern |
| **Pattern (non-active, from PTN select)** | |
| [PTN] + [TRIG] + [REC] / [STOP] / [PLAY] | Copy / Paste / Clear selected pattern (works across banks) |
| **Track sequence (in Grid Rec)** | |
| [FUNC] + [REC] / [STOP] / [PLAY] | Copy / Paste / Clear active track's sequence |
| **Trig (in Grid Rec)** | |
| Hold [TRIG] + [REC] / [STOP] / [PLAY] | Copy / Paste / Clear trig with plocks |
| Hold multiple [TRIG]s + [REC] | Copy sequence of trigs (preserves relative positions) |
| Hold dest [TRIG] + [STOP] | Paste trig sequence starting from held trig |
| **Sequencer Page (in Grid Rec)** | |
| [PAGE] + [REC] / [STOP] / [PLAY] | Copy / Paste / Clear active page |
| **Preset** | |
| [TRK] + [REC] / [STOP] / [PLAY] | Copy / Paste / Clear active track's preset |
| **Parameter page** | |
| [PARAM] + [REC] / [STOP] / [PLAY] | Copy / Paste / Clear parameter page (e.g. [SYN] + [REC]) |
| **FX & Mixer** | |
| [FUNC] + [FX] + [REC] / [STOP] | Copy / Paste Send FX page |
| [FUNC] + [MOD] + [REC] / [STOP] | Copy / Paste Mixer page |
| **Voice Setup** | |
| [FUNC] + [REC] / [STOP] (in Voice menu) | Copy / Paste voice settings |
| **Song Edit** | |

## Community-Discovered Techniques

| Shortcut / Technique | Action |
|---|---|
| Euclidean randomization | FUNC+AMP → EUC on → hold AMP, release FUNC → press YES |
| Key Tracking pitch negation | Key Tracking depth -8.00 on osc pitch negates built-in key tracking |
| Note Edit page navigation | FUNC+LEFT/RIGHT in Note Edit jumps across pages faster than PAGE |
| Comb filter as short delay | Set comb frequency very low; feedback and filter to taste |
| Pre-comb amp envelope via LFO | Square LFO, speed 0, use fade-in for attack shaping before comb |
| Transient drum kit via Key Tracking | Map 62 transients to notes with depth 16 (or 64 for kick/snare/hat spread) |
| Track layering + lock trigs | Layer track N→N-1; put lock trigs on receiving track at different length |
| Velocity mod BW filter flip | Velocity Mod on BW.RT just above threshold flips pre/post on harder hits |
| Comb filter in-tune values | FUNC + turn freq knob snaps to ~12 increments (in-tune at 100% key tracking) |
| Init patch attack time | Default init attack ~8; set to 0 for maximum punch |

## MIDI Track Differences

MIDI tracks share all sequencer shortcuts (copy/paste/clear, plocks, transpose) with audio tracks.
Exceptions: retrigs are unavailable; MIDI presets cannot be added to the Preset Pool (no preset locks);
SYN/FLTR/AMP pages expose Channel, Program, Aftertouch, and CC parameters instead of synth engine params.
